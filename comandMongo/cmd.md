# Aqui estão os códigos para trabalhar com o Mongo via terminal

## Inicializar:

```
 mongosh

```

## Visualizar databases:

```
 show databases;
```

## Acessar database:

```
 use namedatabase;
```

## Visualizar collections(Tables):

```
 show collections;
```

## Exibir campos/valores da collection:

```
 db.nameCollection.find()
```
